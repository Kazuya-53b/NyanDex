# NyanDex
```
■ サービス概要
  NyanDexは猫好きに癒しを与える図鑑アプリ
  ログインユーザーは自身の飼い猫のプロフィールを登録できる
  タグ検索で様々な猫を検索できる

■ このサービスへの思い・作りたい理由
  飼い猫をもっと多くの人に知ってもらいたい、また色々な猫を見ることで癒されたいという思いから

■ ユーザー層について
  猫を飼っている人や、猫好きの人
  色々な猫を見たいという気持ちがあるとおもったため

■ サービスの利用イメージ
  ユーザーは飼い猫の情報を登録し、他の猫好きと共有できる
  タグを使って猫の種類や色、柄を簡単に検索し、気になる猫を見つけることができる

■ ユーザーの獲得について
  SNSでの発信を予定

■ サービスの差別化ポイント・推しポイント
  飼い猫の情報を簡単に登録でき、さらにタグ機能を使って猫の種類や色、柄を詳細に検索できる点が特徴
  猫のプロフィールなどを見ることができ、画像検索よりも身近に感じられる

■ 機能候補
・MVPリリース時に作っていたいもの
  - ユーザープロフィール登録機能
  - 飼い猫のプロフィール登録機能
  - タグを使った猫の検索機能
  - ブックマーク機能

・本リリースまでに作っていたいもの
  - タグ付け時のオートコンプリート機能
  - 検索条件保存機能

■ 機能の実装方針予定
  Stimulus Autocomplete等を使用し、タグ付け時のオートコンプリート機能実装
  MiniMagick等を使用して、アップロードされた猫の画像を加工し、見やすくする
